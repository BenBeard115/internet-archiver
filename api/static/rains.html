Vardo Coat is a waterproof winter coat for the coldest forecasts. The design is part of the Vardo concept: Rains’ warmest and lightest outerwear series, delivering an aesthetic fusion of sport and tech. The coat’s lightweight performance is built on compact featherless insulation and Rains signature nylon – a crisp fabric with a luxe reverse side coating and soft matte finish.

Vardo Coat features an array of front utility zip pockets and an integrated packable hood. More features include taped seams, a snap-fastened double storm placket, full lining with part mesh detail, Velcro cuff straps, jersey storm cuffs and hardware detail at front.
