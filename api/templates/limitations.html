The Internet Archiver: Limitations

Due to the nature of our archiving process alongside the complexities of modern web pages, not all content on an archived web page will work or display correctly.

Web page archives are simply a snapshot of what was online and accessible at the time it was taken, it is not a 'backup' or copy of a webpage, from which the original web page can be fully restored at a later date.

Listed below are the main things that can't be fully preserved by our application:

  • Links within archived webpages to other pages or sites.

    For example, if you are viewing a BBC news article and there is a link to another article within that.

  • Links inside documents

    (.pdf, .doc, .docx, .xls, .xlsx, .csv documents) do not currently work in the web archive.

  • Interactive Content

    Any content that requires user input to proceed, e.g. tick boxes, login fields, search boxes, any forms.

  • Limited Access Content

    Any content that requires a login to be accessed.

  • Some Navigational Features

    Things such as drop down menus, navigation search bars and so on.

  • Image Galleries

    Large images, or collections of images can sometimes be difficult to capture correctly.

  • Embedded Content

    E.g. google maps, media content such as videos, social media content.

  • Flash Animations, Games or Streaming Media
